#트렌드 평형 분석기

여론·관심 트렌드를 정·반·합 평형 엔진으로 **국면 진단**한다.
(예측이 아니라 '지금 어느 국면인가'를 진단 — 정직하고 실현 가능)

## 핵심
- **정(양적)**: 트렌드 상승·생동 신호(최근 기울기, 가속, 급상승 검색어)
- **반(음적)**: 식음·포화 신호(고점대비 하락, 꺾임, 포화, 변동성)
- **합**: 정·반의 긴장을 종합해 국면 점수
- **평형 도달** → 국면 확정 / **팽팽(평형 안 됨)** → "전환점·불확실, 판단 유보"
  - 억지로 답 안 냄. 불확실하면 불확실하다고 정직하게.
- **LLM 속기사(선택)**: 엔진의 판단을 문장으로 옮김. 판단은 엔진, LLM은 번역.

## 국면 라벨
상승 / 완만한 상승·초입 / 정점·전환점(진입 주의) / 완만한 하락·포화 /
하락 / 횡보 / 전환점·불확실(판단 유보)

## 검증됨
상승형→상승, 정점형→정점·전환점, 하락형→하락 정확히 진단.
(짧은 노이즈 데이터는 방향 판독이 어려우나, 실제 구글 트렌드는 길어서 안정적)

## 데이터
- **구글 트렌드(pytrends)**: API 키 불필요. 키워드 시계열 + 관련 검색어.
  ⚠️ 이 저장소 환경은 네트워크 제한 → 로컬에서 `pip install pytrends` 후 실행.
- **가상 데이터**: 로직 테스트용(상승/정점/하락/횡보 패턴).

## pkl 저장/불러오기
- 수집한 트렌드 스냅샷을 `data/trend_snapshots.pkl`에 자동 저장.
- 사이드바에서 다운로드/업로드.
- **모바일 업로드**: 모든 파일 형식 허용 → 코드에서 pkl만 자동 판별해 복원
  (pkl이 안 보이는 모바일 파일창 문제 회피 — 임용 앱과 동일 방식).

## 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```
키워드에 "생성형 AI" 넣고 분석 → 국면 확인.

## 구조
- `core/core_base.py` : 코어 추상 + 평형 기반 클래스
- `core/equilibrium.py` : 평형 엔진(정반합 순환·생존값·되먹임)
- `core/trend_cores.py` : 트렌드 국면 진단 정·반·합 + 신호 추출 + 국면 라벨
- `core/trend_data.py` : pytrends 연결 + pkl 저장/로드/판별
- `app.py` : Streamlit UI
